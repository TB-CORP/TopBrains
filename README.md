# TopBrains
TopBrains is a project of TB-CORP LLC representing an infrastructure of technological solutions with the main purpose - production and distribution of premium courseware to the end user. This repository includes a set of articles of different categories on various topics concerning the TopBrains project, documentation, informative articles, knowledge base, SOPs, etc.
